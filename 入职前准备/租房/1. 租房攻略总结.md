# **租房攻略总结**

## **选房注意**

- 公寓/住宅、电梯房/步梯房、避免顶楼一楼、阳台、采光、水电、燃气、电器、周边环境商圈
- 预算不宜超过月收入1/4 ~ 1/3、避免隔断房、避免合租、提前了解物业费、水电费



## **看房指南**

1. 检查采光和通风情况
2. 检查水电、燃气、空调、热水器、油烟机、洗衣机
3. 检查墙面裂缝、渗水、地面凭证
4. 检查门窗是否完好，锁具是否安全
5. 检查周边交通，步行距离是否合理
6. 检查生活配套，是否有超市、菜市场、医院
7. 检查周围是否有噪声，比如工厂、酒吧



## **签约指南**

1. 确认房东身份，要求出示房产证和身份证，核实其所有权和出租权，认真确认房产证地址和面积是否相同

2. 入住前检查房屋设施，记录损坏写进合同

3. 认真阅读合同条款

   - 确定租金、支付方式（押一付三等）、水电费、物业费
   - 确定租期与续租条件
   - 明确房屋设施损坏的维修职责

   > 合同内容
   >
   > 基本信息：合同中应明确出租人和承租人的姓名、身份证号码、联系方式、住址等，确保双方身份清晰可查。
   > 房屋基本情况：详细描述租赁房屋的具体位置、面积、户型、装修及设施状况等，避免后续因房屋状况产生纠纷。
   > 租赁期限：确定租赁的起止时间，同时约定是否可续租及续租条件。
   > 租金及押金：明确租金金额、支付方式（如押一付三、押二付一等）、支付周期，以及押金数额、退还条件等，保障双方经济权益。
   > 费用承担：明确水费、电费、煤气费、物业管理费等费用的承担方式，并在合同中列明租住前的各项数字，以区分责任。
   > 房屋用途：规定承租人对租赁房屋的使用范围，防止承租人擅自改变用途。
   > 维修责任：分清出租人和承租人在房屋及其附属设施维修方面的责任，确保房屋正常使用。
   > 违约责任：约定双方在违反合同约定时应承担的责任，促使双方严格履行合同。
   > 争议解决方式：一般可以选择协商、仲裁或诉讼等方式解决可能出现的纠纷。

4. 签约完成后保留合同副本、押金收据、转账记录等



## **平台**

| 品牌/平台         | 定位/类型            | 核心优势                                                     | 潜在问题/劣势                                                |
| :---------------- | :------------------- | :----------------------------------------------------------- | :----------------------------------------------------------- |
| **自如**          | 品牌公寓式租房平台   | 房源丰富，房屋风格统一，装修较好，提供保洁、维修等服务，租房流程便捷。 | 租金相对较高，部分房源质量不达标，服务费用增加租客负担。     |
| **泊寓**          | 集中式品牌公寓       | 房屋集中管理，有公共活动区域，提供一定生活配套。             | 水电费较贵，隔音效果差。                                     |
| **相寓/我爱我家** | 中介式租房平台       | 房源种类多，涵盖整租、合租等多种形式，中介服务经验丰富。     | 可能存在中介乱收费、房源信息不真实等问题。                   |
| **乐有家**        | 房地产中介服务型平台 | 有大学生租房免佣金优惠及一系列优惠特价，有专业经纪团队，服务优质，线上线下一体化。 | 中介费较高，部分房源信息需进一步核实。                       |
| **链家**          | 中介式租房平台       | 房源信息较为全面，有专业的经纪人提供服务，品牌知名度高。     | 中介费用较高，部分经纪人专业素质参差不齐。                   |
| **贝壳**          | 综合房产服务平台     | 房源信息整合全面，包括租房、买房等多种房产交易，提供VR看房等便捷服务。 | 平台上的中介服务质量可能参差不齐，部分房源信息更新不及时。   |
| **小红书**        | 社交电商平台         | 用户群体庞大且活跃，社交电商模式独特，品牌影响力强，内容丰富多样。 | 盈利模式不明晰，监管不确定性，内容质量参差不齐，过度商业化风险。 |
